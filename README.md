# ofz-yield-forecasting
Прогнозирование доходности ОФЗ на основе макроэкономических и рыночных факторов с использованием методов машинного обучения (XGBoost). Для проекта Cbonds.
Исследование посвящено прогнозированию направления изменения доходности 10-летних ОФЗ на горизонте 1 месяца с применением алгоритма градиентного бустинга (XGBoost).
Данные:
- Период: январь 2014-декабрь 2025 (144 наблюдения)
- Источники: Cbonds, ЦБ РФ, Росстат, FRED
- Целевая переменная: изменение доходности 10Y ОФЗ (Δy)
## Предикторы
| Переменная | Описание | Источник |
|---|---|---|
| key_rate | Ключевая ставка ЦБ РФ | ЦБ РФ |
| cpi_mom | Инфляция ИПЦ м/м | Росстат |
| cpi_yoy | Инфляция ИПЦ г/г | Росстат |
| usd_rub | Курс USD/RUB | Cbonds |
| brent | Нефть Brent | Cbonds |
| spread_10y_2y | Спред 10Y–2Y ОФЗ | Cbonds |
| rgbi | Индекс RGBI | Cbonds |
| ust10y | US Treasury 10Y | FRED |


Результаты
| Модель | MAE | Accuracy |
|---|---|---|
| Наивный прогноз | 0.363 | 47.0% |
| Lasso | 0.544 | 47.8% |
| XGBoost (базовый) | 0.438 | 52.2% |
| XGBoost (с лагами) | 0.470 | 55.6% |

Структура репозитория
- `data/` — итоговый датасет
- `notebooks/` — Jupyter ноутбук с полным кодом
- `plots/` — графики EDA и SHAP-анализа

Библиотеки: pandas, numpy, scikit-learn, xgboost, shap, matplotlib, seaborn
